Карта мира

 

Карта мира разделена на 25 секторов, в 24 из которых возможно перейти (Восточный остров пока не открыт для посещения). Периодически карта расширяется, завоевываются и исследуются новые сектора.

Перемещение осуществляется с помощью стрелок, расположенных в правом нижнем углу окна. Время перемещения между секторами по горизонтали и вертикали составляет по 120 секунд, по диагонали – 169 секунд. Ускорить передвижение по карте можно с помощью транспорта.

Обозначения на карте

Индикатор текущего местоположения на карте. Обозначает сектор, в котором находится герой.
Показывает сектор, в котором находится текущее задание гильдии наемников. К нему надо перейти для выполнения задания после его получения в гильдии наемников.
Обозначает сектора с гильдией наемников (всего – 3). Становится активным после выполнения (или провала) задания. К одному из них надо перейти для отчета о задании.
